Highlights

它解决的是每天都会遇到的小摩擦。

常见文件管理器把功能分散在菜单、终端、压缩工具和远程客户端里。Mac Explorer 把这些动作收拢成一个连续的工作流，让你少切换、少等待、少猜文件在哪里。

01

看得更完整

列表和网格视图都支持排序、分组、过滤、Git 标记、评分与元数据预览，文件状态不用跳到别的工具里确认。

02

找得更自然

文件名搜索、FTS5 全文索引、AI 标签和图片内容理解一起工作。你可以按名字找，也可以按“人物、场景、地点、关键词”找。

03

操作更连贯

复制、移动、压缩、解压、SFTP 上传、Quick Look 预览和右键快捷操作都围绕同一个窗口设计，批量任务也有清晰进度。

Features

功能够强，但不用把人推向技术细节。

Mac Explorer 把高级能力做成普通操作：你仍然是在浏览文件，只是搜索更聪明、远程目录更像本地、压缩包更像文件夹、Git 状态更容易被看见。

AI本地优先

AI 图片与内容理解

自动分析图片中的人物、场景、地点与文字信息，为文件生成可搜索标签。找照片不必只靠文件名，也不必把隐私交给云端。

FTS快速定位

文件搜索与智能建议

首页可以像命令面板一样输入路径或关键词；目录内搜索会优先使用索引，也能回退到文件系统递归搜索。

Git图标标记

Git 状态一眼可见

新增、修改、未跟踪等状态会叠加在文件图标上。浏览仓库时不用打开终端，也能知道哪些文件正在变化。

SFTP远程目录

SFTP 连接与远程编辑

保存服务器信息后即可浏览远程目录，支持密码或密钥认证、文件上传下载与远程编辑，体验更接近本地文件夹。

ZIP无需绕路

压缩、解压与归档浏览

支持 ZIP、TAR、GZ、7Z 等常见格式。压缩任务在后台运行，压缩包也可以像文件夹一样进入查看。

QL原生能力

Quick Look 与元数据面板

空格快速预览沿用 macOS 习惯，侧边信息面板展示大小、类型、修改时间、扩展属性、评分和图片预览。

DRAG批量操作

多选拖放与冲突处理

框选、Shift 连选、Cmd 点选、批量拖放都按桌面习惯设计。复制或移动遇到冲突时，会提示你选择处理方式。

UI双主题

毛玻璃窗口与紧凑工具栏

侧边栏、菜单、预览面板和任务面板都采用玻璃质感。浅色与深色主题不是简单反色，而是为长时间使用重新调过层次。

Compare
