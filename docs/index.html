功能概览

比 Finder 多走一公里

Mac Explorer 不是 Finder 的替代品，而是一个增强层。它保留了 macOS 的原生质感， 同时引入了 Windows 文件管理器中那些 macOS 缺失已久的能力。

AI 智能标签与搜索

自动提取当前目录中的高频词条并进行智能分组。支持文本搜索和词云可视化，帮助快速定位内容。本地 AI 解析图片，无需联网。

Git 状态集成

文件和目录图标叠加 Git 状态标记——新增（绿）、修改（橙）、未跟踪（灰），在文件浏览器中即可掌握仓库状态，无需打开终端。

压缩解压 · 直接预览

内置 ZIP、TAR、GZ、7Z 等多格式支持。无需解压即可直接浏览压缩包内容，像操作普通文件夹一样预览内部文件。支持分卷压缩和密码保护。

网格 / 列表双视图

一键切换网格图标视图与详细信息列表视图。列表支持列宽拖拽调整、多列排序，网格支持框选多选和拖放操作。

多选拖放与批量操作

支持框选、Shift 连选、Cmd 点选多种选择模式。拖放移动、复制、剪切粘贴，冲突检测自动提示处理策略。

快速预览面板

侧边栏预览窗格支持图片预览、文件元数据、扩展属性查看。无需打开文件即可确认内容。

自定义"打开方式"

为任意文件类型配置打开方式应用列表，支持自定义应用路径和参数，替代系统默认应用关联。

后台任务面板

文件操作（复制、移动、压缩）在后台执行，右下角悬浮面板实时显示进度。支持多任务并行和失败重试。

文件评分系统

支持 1–5 星文件评分，评分数据持久化存储。列表和网格视图均可直接操作评分，支持按评分排序。

深色模式 · 毛玻璃 · 亚克力

完整亮/暗主题切换，大量使用 macOS 原生毛玻璃材质（Vibrancy）和 Windows 11 亚克力半透明效果。

设计哲学

处处精致

从毛玻璃侧边栏到亚克力材质右键菜单，每个像素都经过仔细推敲。不是 Finder 的皮肤，而是一次彻底的视觉重构。

侧边栏、右键菜单、预览面板、设置弹窗——所有浮层均采用 backdrop-filter: saturate(180%) blur(20px)，Mac 的 Vibrancy 与 Win11 的 Acrylic 在此融合。

技术栈

现代技术构建

Mac Explorer 基于 .NET 10 和 MAUI Blazor 构建，充分利用 macOS 原生能力。

.NET 10 + MAUI Blazor

使用最新的 .NET 10 跨平台框架，MAUI Hybrid 方案实现原生窗口 + Web UI 的完美融合。

MASA Blazor 组件库

基于 Material Design 的企业级 Blazor 组件库，提供对话框、菜单、输入等标准化 UI 交互。

原生 macOS API

通过 P/Invoke 调用 AppKit、AppleScript、Finder 扩展等原生能力，实现图标提取、废纸篓操作等底层功能。
